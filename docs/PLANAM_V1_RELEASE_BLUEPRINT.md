# 1. Миссия PLANAM

PlanAm существует не для хранения рецептов.

PlanAm существует не для подсчёта калорий.

PlanAm существует не для ведения списков покупок.

## Настоящая миссия

```text
Убрать ежедневную усталость от принятия решений о питании семьи.
```

Каждый день миллионы людей решают:

- что приготовить;
- что купить;
- что есть дома;
- что полезно;
- что понравится семье.

PlanAm должен принимать большую часть этих решений за пользователя.

---

## Эволюция продукта (V1 → будущее)

PlanAm начинается как **помощник по питанию семьи**.

Постепенно становится **ежедневным центром бытовых решений семьи**.

- **Питание** остаётся ядром продукта.
- **Покупки, остатки, здоровье и семья** расширяют ежедневную ценность.

Это необходимо для: единого семейного списка покупок, детских и зоотоваров, будущей доставки.

# 4. Роль AI

## Главный принцип

AI должен работать в фоне.

Пользователь не должен думать:

```text
Надо открыть AI
```

Пользователь должен видеть:

```text
Готовое решение
```

## Бесплатный AI

Доступен всем.

| Возможность | Сценарий |
|-------------|----------|
| AI меню | Составить меню |
| AI покупки | Меню → покупки; классификация типа и категории |
| AI остатки | Что приготовить из того, что есть |
| AI замена блюда | Не хочу это блюдо → получить замену |
| AI семейные предпочтения | Учитывать всю семью автоматически |
| AI категоризация покупок | Рецепт / дом / ребёнок / питомец → каноническая категория |

### AI и типы покупок

AI различает **тип покупки** (не только ингредиент меню):

| Тип | Примеры | Категория |
|-----|---------|-----------|
| Покупка для рецепта | Курица, молоко, помидоры | Мясо и птица · Молочные · Овощи… |
| Покупка для дома | Порошок, бумага, мыло | Быт и уборка |
| Покупка для ребёнка | Подгузники, смесь | Детские товары |
| Покупка для питомца | Корм для кошки | Для питомцев |

```text
Курица → Мясо и птица
Молоко → Молочные продукты
Подгузники → Детские товары
Корм для кошки → Для питомцев
Порошок → Быт и уборка
```

## PRO AI

**AI Nutritionist** — главный драйвер монетизации.

Не отдельный продукт. Часть раздела «Здоровье».

PRO получает:

```text
AI-анализ питания
AI-рекомендации
AI-планы
AI-прогноз
AI-нутрициолога
```

# 6. Главный экран PlanAm

Сердце продукта.

Не список функций. Не дашборд. Не админка.

**Требование:** первый экран должен помещаться почти полностью без скролла.

## Динамический Hero

Hero показывает **следующий наиболее полезный шаг** (см. [PLANAM_V1_FINAL_VISION.md](./PLANAM_V1_FINAL_VISION.md)).

**Не привязан к ужину.** По умолчанию — фото **ближайшего приёма пищи**; при более важном действии Hero перестраивается (меню / покупки / здоровье).

## Структура (wireframe — пример: следующий приём пищи)

```text
Доброе утро, Иван 👋

[Фото ближайшего приёма пищи]

Курица терияки · обед
30 минут · 540 ккал

[Приготовить]

━━━━━━━━━━

🛒 Купить          7 товаров
📦 Остатки         43 продукта
❤️ Здоровье        в норме

━━━━━━━━━━

[Открыть меню]
[Список покупок]

✨ Спросить PlanAm
```

**Альтернативный Hero (покупки важнее):**

```text
Нужно купить 12 товаров
[Открыть список]
```

**Главный визуальный объект по умолчанию:** еда. Исключения — см. Final Vision и Image Strategy.

**Иерархия действий:**

1. Primary CTA Hero (Приготовить / Создать меню / Открыть список / Подробнее)
2. Три статуса (покупки / остатки / здоровье)
3. Вторичные CTA (меню, список)
4. AI-вход (одна точка, не отдельная вкладка)

# 7. Экран «Сегодня»

**Ответ:** что мы едим сегодня?

## Структура

```text
Завтрак    [Карточка]
Обед       [Карточка]
Ужин       [Главная карточка]
```

## Иерархия действий на карточке блюда

| Приоритет | Действие |
|-----------|----------|
| Primary | **Приготовить** |
| Secondary | Рецепт · Заменить · В покупки |
| Скрытое / редкое | Удалить (не primary) |

Удаление не должно быть главным действием.

# 8. Экран «Покупки»

**Ответ:** что нужно купить?

**Ощущение:** «Я собираюсь в магазин» — единый семейный список, не только ингредиенты из меню.

## Семейные покупки

PlanAm хранит **единый список покупок семьи**.

Список включает:

```text
Ингредиенты из меню
Ручные покупки
Домашние товары
Товары для детей
Товары для животных
```

Пользователь **не должен** использовать отдельное приложение для бытовых покупок.

Любой adult может добавить позицию вручную; позиции из меню попадают в тот же список автоматически.

## Категории (канон V1)

```text
🥬 Овощи и зелень
🍎 Фрукты и ягоды
🥩 Мясо и птица
🐟 Рыба и морепродукты
🥛 Молочные продукты
🥚 Яйца
🍞 Хлеб и выпечка
🍚 Крупы, макароны и бакалея
🧂 Специи, соусы и масла
🥤 Напитки
🧻 Быт и уборка
🧸 Детские товары
🐾 Для питомцев
📦 Другое
```

**14 категорий.** AI и ручной ввод обязаны маппить позицию в одну из них.

## Запрещённая категория

```text
Продукты
```

Не использовать. Универсальная корзина «Продукты» не отображается пользователю и не используется в данных V1.
