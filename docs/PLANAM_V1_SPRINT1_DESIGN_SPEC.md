### Раскладка 5 вкладок (375px — iPhone SE)

```text
│←16px→│ Tab1 59px │ Tab2 59px │ CENTER 75px │ Tab4 59px │ Tab5 59px │←16px→│
        Сегодня     Покупки      ПланАм       Здоровье    Профиль
```

На **320px** (узкий Android): `px-0.5`, label **10px**, иконка **18px**, центр **68px** — все 5 подписей помещаются без truncate (самая длинная: «Здоровье» 7 букв).

## 1.2 Размеры и отступы

| Элемент | Значение | Примечание |

| Gap между вкладками | 2px (`gap-0.5`) | |

**Триггер:** `unchecked_count >= 8` ИЛИ `hours_to_next_meal > 3` И `unchecked_count >= 3` ИЛИ explicit «перед магазином» window (17:00–19:00).

```text
┌──────────────────────────────────────┐
│  Добрый день, Иван 👋                │
│  ┌────────────────────────────────┐  │
```

# ЗАДАЧА 4 — 3-секундный тест

Оценка: ✅ понял · ⚠️ частично · ❌ нет

| Hero state | Что готовить | Что купить | Что дома | Здоровье | Итог |
|------------|--------------|------------|----------|----------|------|
| Первый запуск | ⚠️ «составим меню» | ✅ пусто | ✅ «после готовки» | ⚠️ «заполни профиль» | ✅ pass |
| Нет меню | ✅ CTA меню | ✅ 0 товаров | ✅ остатки row | ✅ row | ✅ pass |
| Есть меню | ✅ фото+название | ✅ 7 товаров | ✅ 43 продукта | ✅ в норме | ✅ pass |
| Покупки важнее | ⚠️ не в hero | ✅ **12 товаров** hero | ✅ row | ✅ row | ✅ pass |
| Инсайт здоровья | ⚠️ не в hero | ✅ row | ✅ row | ✅ **в hero** | ✅ pass |
| Всё в порядке | ✅ следующий приём | ✅ row | ✅ row | ✅ row | ✅ pass |

**Почему ⚠️ не провал:** 3 статуса всегда на экране — даже если Hero про покупки, строка 🛒📦❤️ закрывает остальные вопросы за 3 сек.

**Единственный риск провала:** ShellHeader + высокий Hero + нет статусов на экране → **mitigation:** скрыть header на `/`, compact hero на SE.

# ЗАДАЧА 5 — Telegram UX

## 5.1 Элементы

| Элемент | Действие Sprint 1 |
|---------|-------------------|
| **Telegram Header** | Не дублировать: на `/` без App shell title; `BackButton.hide()` на root tabs |
| **Telegram Back** | Показывать только на 2+ уровне; `TelegramBackBridge2026` + `returnTo` |
| **Telegram Theme** | Синхронизировать `bg_color` ↔ `pa-canvas`; `colorScheme` dark/light |
| **Safe Areas** | `safe-area-inset-top` в greeting padding; bottom в nav |
| **Bottom Navigation** | Fixed; не перекрывать контент (offset в main) |

## 5.2 Скрывать / использовать / мешает

| Скрывать | Использовать | Мешает |
|----------|--------------|--------|
| 🏠 в header на `/` | TG theme sync | Двойной header (TG + App) |
| ShellHeader на `/` | `expand()` при onboarding | Legacy quick actions grid |
| Back на tab roots | Haptic на CTA (P1) | 4 tabs без ПланАм (as-is) |
| Баннер PRO на `/` | MainButton — **не** для Sprint 1 | Floating AI chat entry |

## 5.3 Пять вкладок на маленьких экранах

**Вердикт: можно оставить 5 вкладок.**

| Ширина | Стратегия |
|--------|-----------|
| 320px | `px-0.5`, icons 18px, labels 10px, center circle 40px |
| 375px | Spec по §1.2 |
| 390+ | Стандарт |

**Не обрезается:** «Здоровье», «Профиль» — короче 8 символов.

**Запасной план (только если QA fail на 320px):** сократить «Здоровье» → «Здоров.» в micro-font — **не в Sprint 1**, только по данным QA.
