# PLANAM V1 — Growth Model

**Версия:** 1.0  
**Дата:** 2026-06-03  
**Принцип:** не придумывать механики ради механик — только то, что усиливает core loop

---

# Core loop (основа удержания)

```text
Открыл ПланАм → увидел ужин + что купить
  → купил / приготовил
    → отметил
      → завтра снова открыл
```

Вирусность V1 строится на **совместной пользе**, не на геймификации.

---

# 1. Почему пригласит супруга

| Мотив | Как продукт это даёт |
|-------|----------------------|
| **Разделить покупки** | Один список; супруг открывает Покупки и отмечает без пересылки |
| **Разделить готовку** | Общее меню на Сегодня — кто готовит, видит тот же план |
| **Меньше споров** | «Что на ужин» решено на ПланАм до 18:00 |
| **Свои цели, общий стол** | Профили разные; меню — компромисс AI |

**Триггер приглашения:** после первой успешной недели admin видит CTA «Пригласи супруга — пусть видит список покупок».

**Механика V1:** семейное приглашение (Telegram link). Реферальная программа — **P2**, только после данных по retention.

---

# 2. Почему пригласит родителей

| Мотив | Как продукт это даёт |
|-------|----------------------|
| **Помочь с детьми** | Virtual member: аллергии, возраст, детское питание |
| **Общий ужин в гости** | Меню на выходные учитывает +2 порции (P1 life scenario) |
| **Забота о пожилых** | Virtual profile: ограничения, мягкие блюда |
| **Не учить приложение** | Родитель с Telegram — invite; без — admin ведёт профиль |

**Триггер:** после добавления ребёнка — «Добавь бабушку, чтобы она видела меню на выходные».

**Не делаем в V1:** отдельный «родительский» продукт или второй onboarding.

---

# 3. Почему пригласит друзей

| Мотив | Реалистичность V1 |
|-------|-------------------|
| Совместный ужин / барбекю | Слабый для daily app — **не primary growth** |
| «Попробуй, классное меню» | Сарафан после wow-момента (первое меню за 5 мин) |
| Реферальный бонус | **P2** — только если есть monetization + retention |

**V1 фокус:** друзья — не целевой канал. Рост через **семью**, не через social graph.

**Исключение:** share рецепта / «меню на неделю» — P2, если появится export.

---

# 4. Почему не удалит через месяц

| Причина | Механизм |
|---------|----------|
| **Привычка утра** | ПланАм = 3 секунды: ужин + покупки + здоровье |
| **Список всегда актуален** | Меню → покупки автоматически |
| **Меньше когнитивной нагрузки** | Не думать «что готовить» |
| **Прогресс здоровья** | Калории/вода видны без усилий |
| **Семья завязана** | Удаление = сломать общий список для супруга |
| **Остатки** | «43 продукта дома» — ценность накопленных данных |
| **Сезонность жизни** | Сценарии: болезнь, праздник — продукт адаптируется |
| **PRO (если купил)** | AI Nutritionist даёт персональный совет |

**Anti-churn V1:** не push-спам, а **полезный статус на главной** каждый день.

# 5. Семейные приглашения (V1)

| Элемент | Описание |
|---------|----------|
| Entry | Профиль → Семья → Пригласить |
| Канал | Telegram bot deep link `invite_{token}` |
| Copy | «Присоединяйся к нашему меню и списку покупок» |
| Friction | 3 шага: телефон → отправка → принятие |
| Post-accept | Family mode auto; общее меню visible |

---

# 6. Совместные покупки (V1)

| Элемент | Описание |
|---------|----------|
| Не «корзина магазина» | Список дел в магазине |
| Sync | Real-time при открытии вкладки Покупки |
| Главная | «7 товаров» — мотивация открыть перед магазином |
| Будущее | V2: кнопка «Заказать доставку» на том же списке |

---

# 7. Совместное меню (V1)

| Элемент | Описание |
|---------|----------|
| Генерация | Один раз на семью |
| Замена | Любой adult; AI учитывает всех |
| Сегодня | Единый экран для всех |
| Уведомление | «Меню на неделю готово» (P1) |

---

# 8. Реферальная система

**Статус:** P2 (после первых пользователей)

**Условие запуска:**
- D7 retention > порога
- Семейный invite conversion известна
- Monetization (PRO) стабильна

**Не в V1:** баллы, уровни, «приведи 5 друзей».

---

# 9. Метрики роста V1

| Метрика | Цель |
|---------|------|
| D1 open ПланАм | > 60% активных |
| Family invite sent | > 30% admin после 7 дней |
| Family invite accepted | > 50% sent |
| D30 retention | > 25% (личный); > 40% (family) |
| Shopping list open/week | > 2x для family |

---

*Рост V1 = семья + ежедневная полезность главного экрана, не вирусные механики.*
