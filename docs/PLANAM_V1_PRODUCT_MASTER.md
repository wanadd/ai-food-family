# Продукт в одном абзаце

PlanAm убирает ежедневную усталость от решений о питании семьи: **один экран ПланАм** отвечает «что готовить, что купить, что дома, что по здоровью»; AI работает в фоне; семья делит меню и покупки; V1 готовит архитектуру к доставке без переписывания.

---

# Принцип: эволюция продукта

PlanAm начинается как **помощник по питанию семьи**.

Постепенно становится **ежедневным центром бытовых решений семьи**.

- **Питание** остаётся ядром продукта.
- **Покупки, остатки, здоровье и семья** расширяют ежедневную ценность.

Необходимо для: единого семейного списка покупок, детских и зоотоваров, будущей доставки.

---

# Принцип: единый семейный центр покупок

PlanAm является **единым семейным центром покупок**, а не только генератором продуктовых списков из меню.

В одном списке семьи:

```text
Ингредиенты из меню
Ручные покупки
Домашние товары
Товары для детей
Товары для животных
```

Пользователь не должен использовать отдельное приложение для бытовых покупок.

**14 категорий** (от 🥬 Овощи до 🐾 Для питомцев). Категория **«Продукты»** запрещена.
