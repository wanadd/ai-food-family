# PLANAM V1 — AI Journey

**Версия:** 1.0  
**Дата:** 2026-06-03  
**Главный принцип:** AI не отдельный чат. AI = готовые решения на экранах + одна точка «Спросить PlanAm».

---

# Карта касаний AI

```text
Onboarding → генерация меню (AI)
     ↓
ПланАм → insight + «Спросить PlanAm»
     ↓
Сегодня → замена блюда (AI)
     ↓
Покупки → sync из меню (AI aggregation)
     ↓
Здоровье → совет дня (free) / AI Nutritionist (PRO)
     ↓
Профиль → семейные предпочтения (AI при генерации)
```

**Нет вкладки «AI».** `/wellness/chat` — второй уровень для PRO, не entry point.

---

# Часть 1. Бесплатный пользователь

## 1.1 Когда сталкивается с AI

| Момент | Что происходит | Где |
|--------|----------------|-----|
| **Первое меню** | AI генерирует неделю по профилю | `/plan/generate`, onboarding |
| **Открытие ПланАм** | Видит готовое блюдо (результат AI, не кнопку AI) | `/` |
| **Замена блюда** | «Не хочу это» → AI предлагает альтернативу | `/plan/today` → replace |
| **Список покупок** | Позиции из меню собраны AI | `/home/shopping` |
| **Семья** | Меню учитывает всех без ручного merge | family generate |
| **Спросить PlanAm** | Явный вопрос пользователя | Sheet с `/` |

Пользователь **не ищет AI** — AI уже сработал до открытия экрана.

## 1.2 Какие решения получает (free)

| Решение | Формат |
|---------|--------|
| Меню на неделю | Карточки на Сегодня |
| Замена блюда | 1–3 варианта в replace flow |
| Список покупок | Группировка по категориям |
| Учёт аллергий | Исключение до показа блюда |
| Краткий ответ | «Спросить PlanAm» — 2–4 предложения |

## 1.3 Когда понимает ценность

| Aha-момент | Почему |
|------------|--------|
| **Меню за 5 минут** | Onboarding → hero на ПланАм |
| **«Не думал что готовить»** | Утро: открыл — ужин уже выбран |
| **Замена за 10 сек** | Не нравится → другое блюдо сразу |
| **Список сам собрался** | Не вводил вручную 20 позиций |
| **Ребёнок с аллергией** | Меню без аллергена без напоминаний |

**Метрика aha:** первый «Приготовить» в течение 48ч после регистрации.

# Часть 2. PRO пользователь

## 2.1 Когда сталкивается с AI Nutritionist

| Момент | Триггер |
|--------|---------|
| Открытие Здоровья | PRO block виден сразу (не чат) |
| Tap «AI-анализ» | Развёрнутый блок на `/wellness` |
| «Спросить PlanAm» с PRO | Расширенный ответ (нутрициолог tone) |
| Paywall был на free | После upgrade — тот же экран, больше контента |

**Не:** отдельная вкладка «Чат с AI».

## 2.2 Когда получает рекомендации

| Тип | Когда | Где |
|-----|-------|-----|
| Совет дня | Ежедневно при открытии Здоровья | `/wellness` |
| Рекомендация по блюду | После замены / перед готовкой | Сегодня, sheet |
| Рекомендация по неделе | Понедельник / после генерации | Здоровье PRO block |
| Семейная | При family mode | Внутри анализа |

## 2.3 Когда получает прогнозы

| Прогноз | Частота |
|---------|---------|
| Калории на неделю | После генерации меню |
| Белок / дефициты | При открытии Здоровье |
| Вес (если цель) | P2 `/progress` |

Формат: **1 карточка**, не отчёт на 3 экрана.

## 2.4 Когда получает аналитику

| Аналитика | Условие |
|-----------|---------|
| Недельный обзор БЖУ | PRO + ≥3 дня данных |
| Сравнение с целью | PRO + nutrition_goal |
| AI-нутрициолог (чат) | PRO + явный tap «Углубиться» → `/wellness/chat` |

Чат — **углубление**, не стартовая точка.

# Часть 3. «Спросить PlanAm» — единая точка

## Поведение

| Аспект | V1 |
|--------|-----|
| Entry | Только с ПланАм (footer) |
| UI | Bottom sheet, не full-page chat |
| Контекст | Текущее блюдо, список покупок, день |
| Free | Короткие ответы, лимит (AMA или daily cap) |
| PRO | Полные ответы + ссылка «в Здоровье» |

## Примеры запросов

```text
«Чем заменить курицу?»
«Что купить на ужин?»
«Сколько калорий сегодня?»
«Меню для ребёнка без молока»
```

Ответ **действие**, не лекция: «Замени на индейку — вот 2 рецепта» + кнопка.

# Часть 4. AI не как чат — чеклист V1

| ✅ Делаем | ❌ Не делаем |
|----------|-------------|
| Готовое блюдо на hero | «Сгенерировать с AI» как главный CTA |
| Замена в контексте карточки | Отдельная вкладка AI |
| Совет дня в Здоровье | Обязательный чат для value |
| Sheet «Спросить» | Full-screen chat как home |
| PRO блок в Здоровье | PRO = только чат |
| AI в фоне при генерации | «Нажми чтобы включить AI» |
