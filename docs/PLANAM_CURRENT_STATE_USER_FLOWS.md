# PLANAM — пользовательские сценарии (факты)

Дата: 2026-06-03  
Контур описания: UI 2026 (production default). Legacy-ветки указаны где существуют параллельные маршруты.

---

## 1. Открыть рецепт

**UI 2026:**

| Шаг | Экран | Действие |
|-----|-------|----------|
| 1 | `/` или `/plan/today` | Пользователь на главной или «Сегодня» |
| 2 | `/plan/today` | Нажимает «Рецепт» на карточке блюда |
| 3 | `/plan/recipes/[id]?returnTo=/plan/today` | Открывается деталь рецепта |
| Финиш | detail | Просмотр ингредиентов, шагов |

**Альтернативный вход:**

| Шаг | Экран | Действие |
|-----|-------|----------|
| 1 | `/plan/recipes` | Subtab «Рецепты» |
| 2 | `/plan/recipes/[id]` | Tap на карточку каталога |
| Финиш | detail | — |

- **Экранов:** 2–3
- **Кликов:** 2
- **Тупики:** нет (empty state каталога — отдельный путь)
- **Лишние шаги:** нет данных в коде

**Legacy:** `/menu/recipes` → tap → `/recipes/[id]` (modal full-screen)

## 2. Добавить рецепт в меню

**UI 2026:**

| Шаг | Экран | Действие |
|-----|-------|----------|
| 1 | `/plan/recipes/[id]` | «В меню» |
| 2 | Sheet `MenuSlotSheet2026` | Выбор даты, приёма пищи, порций |
| 3 | `/plan/today?saved=1` | API `addRecipeToMenu`, redirect |
| Финиш | today | Блюдо в плане |

- **Экранов:** 2 + 1 sheet
- **Кликов:** 3+ (выбор слота)
- **Тупики:** нет меню → empty на today ведёт на generate
- **Лишние шаги:** sheet обязателен (выбор слота)
