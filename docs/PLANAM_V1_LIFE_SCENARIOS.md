# PLANAM V1 — Life Scenarios

**Версия:** 1.0  
**Дата:** 2026-06-03  
**Формат:** для каждого сценария — что меняется · AI · экраны · рекомендации

---

# 1. Будни

| Аспект | Описание |
|--------|----------|
| **Что меняется** | Быстрые блюда, фиксированное время ужина, минимум шагов |
| **AI** | Генерация с приоритетом «≤30 мин», batch-friendly завтрак/обед |
| **Экраны** | ПланАм → Сегодня → Покупки |
| **Рекомендации** | «Ужин за 25 мин», «купи 5 товаров сегодня», совет по белку |

---

# 2. Выходные

| Аспект | Описание |
|--------|----------|
| **Что меняется** | Более сложные блюда, больше времени на готовку |
| **AI** | Меню недели с «weekend slots» — richer recipes |
| **Экраны** | ПланАм, `/plan` (неделя), каталог |
| **Рекомендации** | «Попробуй новое блюдо», «запасись на 2 дня» |

---

# 3. Гости

| Аспект | Описание |
|--------|----------|
| **Что меняется** | +N порций, учёт предпочтений гостей (ручной ввод P1) |
| **AI** | Пересчёт servings, предложение «меню для 6 человек» |
| **Экраны** | Сегодня (замена), Покупки (доп. позиции), Профиль |
| **Рекомендации** | «Добавь 3 порции к ужину», список закупок ×1.5 |

---

# 4. Праздники

| Аспект | Описание |
|--------|----------|
| **Что меняется** | Особое меню на дату, больше блюд, десерт |
| **AI** | Event-style generation (P2 `/menu/event`; V1 — ручная замена слотов) |
| **Экраны** | `/plan`, каталог, Покупки |
| **Рекомендации** | «Меню на Новый год», за неделю — список покупок |

---

# 5. День рождения

| Аспект | Описание |
|--------|----------|
| **Что меняется** | Торт/десерт, детское меню, аллергии гостей-детей |
| **AI** | Учёт virtual children, sweet slot |
| **Экраны** | Сегодня, Семья (профили детей), Покупки |
| **Рекомендации** | «Торт без орехов для Маши», «детская порция» |

---

# 6. Поездка

| Аспект | Описание |
|--------|----------|
| **Что меняется** | Короткое меню, перекусы, минимум свежих закупок |
| **AI** | Сжатое меню на N дней, non-perishable bias |
| **Экраны** | ПланАм, Покупки, Профиль (пауза уведомлений) |
| **Рекомендации** | «Меню на 3 дня в дорогу», «что не портится» |

---

# 7. Отпуск

| Аспект | Описание |
|--------|----------|
| **Что меняется** | Приложение в фоне; возврат — новое меню с нуля |
| **AI** | «Сброс» плана, лёгкое меню первой недели после отпуска |
| **Экраны** | Onboarding-lite при возврате, ПланАм |
| **Рекомендации** | «Добро пожаловать — меню на неделю за 1 тап» |

---

# 8. Болезнь

| Аспект | Описание |
|--------|----------|
| **Что меняется** | Мягкие блюда, супы, меньше жирного; возможно один sick member |
| **AI** | Restrictions: лёгкая пища, exclude spicy/heavy |
| **Экраны** | Здоровье, Сегодня (замена), Профиль (временное ограничение) |
| **Рекомендации** | «Куриный бульон», «больше жидкости», вода на Здоровье |

---

# 9. Похудение

| Аспект | Описание |
|--------|----------|
| **Что меняется** | `nutrition_goal: lose_weight`, дефицит калорий |
| **AI** | Меню в целевом диапазоне kcal; PRO — анализ недели |
| **Экраны** | Здоровье (калории), Сегодня, Профиль |
| **Рекомендации** | «−300 ккал сегодня», «белок 90г», совет дня |

---

# 10. Набор массы

| Аспект | Описание |
|--------|----------|
| **Что меняется** | Профицит, высокий белок |
| **AI** | Protein-forward recipes, larger portions |
| **Экраны** | Здоровье, Сегодня, Покупки (больше белковых позиций) |
| **Рекомендации** | «+500 ккал», «добавь творог/яйца в список» |

---

# 11. Спортивный режим

| Аспект | Описание |
|--------|----------|
| **Что меняется** | Тайминг приёмов пищи, углеводы до/после нагрузки |
| **AI** | PRO: распределение БЖУ по слотам |
| **Экраны** | Здоровье (PRO), Сегодня |
| **Рекомендации** | «Углеводы за 2ч до тренировки», гидратация |

---

# 12. Пост

| Аспект | Описание |
|--------|----------|
| **Что меняется** | `dietary_restrictions` включает постные правила |
| **AI** | Hard filter: без мяса/молока/яиц по правилам профиля |
| **Экраны** | Профиль → Ограничения, Сегодня, каталог (фильтр) |
| **Рекомендации** | «Постное меню на неделю», замены белка (бобовые) |

---

# 13. Детское питание

| Аспект | Описание |
|--------|----------|
| **Что меняется** | Virtual child member, age_months, мягкие текстуры |
| **AI** | Отдельная логика порций/ингредиентов для child |
| **Экраны** | Семья, Сегодня, Профиль (child nutrition) |
| **Рекомендации** | «Без соли для 1 года», «аллергия на молоко учтена» |
